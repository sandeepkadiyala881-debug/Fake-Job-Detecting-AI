# 🚨 Fake Job Detector

## 📌 Overview

Fake Job Detector is an AI-powered web application that analyzes job postings and detects potential fake or scam-related content.  

The system uses **Groq API** to evaluate job descriptions and generate a scam risk score, helping users stay safe from fraudulent recruitment offers.

This project was developed as a **team project**.

---

## 🎯 Problem Statement

Online job scams are increasing rapidly, targeting students and fresh graduates. Fake job postings often include:

- Unrealistic salary promises  
- Advance payment requests  
- Suspicious links or email domains  
- Urgent language pressure  
- Lack of proper company verification  

There is a need for an intelligent system that can analyze and identify such patterns.

---

## 💡 Solution

Our system:

- Accepts job description input
- Sends data to Groq API for AI analysis
- Processes the response
- Generates a scam probability score
- Displays detailed analysis results

This helps users make informed decisions before applying.

## ✨ Features

- 🔍 AI-based job analysis using Groq API  
- 📊 Scam risk scoring system  
- 📝 Detailed result explanation
